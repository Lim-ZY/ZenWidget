# ZenWidget

ZenWidget is an Android app (with widget) that features customisable quotes and 1-minute actions to tackle doomscrolling.

## Features
- Customisable quotes and actions
- Customisable tasks
- Glass-like app UI and widget background (using (AndroidLiquidGlass)[https://github.com/Kyant0/AndroidLiquidGlass])
- Daily refresh of quotes in widget
- Integrated Pomodoro timer (inspired by (GNOME Solanum)[https://apps.gnome.org/Solanum/])
- Customisable widget size (_Coming soon_)
